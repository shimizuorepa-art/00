# オーナー窓口

ユーザーからの直接指示、相談、雑談、思いつき、運用変更を置く場所。

ここは原作、考証、脚本、編集のどれにも属さない。編集が最初に確認し、必要に応じて各担当へ発注へ変換する。

## 使い分け

- `00_現在の指示.md`: 今後の制作で必ず守る指示
- `01_相談メモ.md`: まだ決定していない相談や迷い
- `02_雑談ログ.md`: 作品の種になりそうな雑談
- `03_採用済みオーナー判断.md`: 採用が確定した判断
- `04_差し込み依頼.md`: 次ラウンドへ割り込ませたい依頼
- `05_オーナー補佐`: ユーザーの相談を受け、指示や発注に変換する補佐役

## 運用ルール

- あなたの直接指示は、部署の作業ファイルより優先する。
- ただの雑談は、即採用せず `02_雑談ログ.md` に置く。
- 採用する場合は、編集またはプロデューサーが `03_採用済みオーナー判断.md` に移す。
- 次のラウンドに影響するものは、軽量発注テンプレートに沿って発注カードや各担当へのサブ発注へ変換する。
