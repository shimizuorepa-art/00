# 運用管理

AIライター室の制作ルール、テンプレート、進行管理を置く。

## 今の基本

- 旧15話は工程再設計前アーカイブへ退避済み。
- 新制作では、章骨子前にキャラ会議を置く。
- 各工程は5つのポジティブな役割を中心にする。
- サブエージェントへの発注は軽量テンプレートで、任せることと寝かせるものを短く渡す。
- 通常回は軽く回し、重要回だけ追加レビューや考証を差し込む。

## 主要ファイル

- `新制作工程.md`: 現行の制作フロー本体。
- `半自動ライティング運用.md`: サブエージェントを含む運用メモ。
- `サブエージェント軽量発注テンプレート.md`: サブへの渡し方を軽く保つテンプレート。
- `章単位制作運用.md`: 章開始、キャラ会議、章末更新の流れ。
- `キャラ会議テンプレート.md`: 章前キャラ会議の出力テンプレート。
- `章末人物更新テンプレート.md`: 章末で人間関係とキャラ設定を更新するテンプレート。
- `章構成メモテンプレート.md`: 章骨子用テンプレート。
- `各話構成メモテンプレート.md`: 各話の発注カード用テンプレート。
- `ラウンド進行管理テンプレート.yaml`: 新工程用の進行管理テンプレート。
- `制作ダッシュボード.yaml`: 全体の現在地。
