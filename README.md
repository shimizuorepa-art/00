# 連続AI小説お試し

連続生成で小説を広げるための作業場。

## 作品仮題

ブループリント

## 現在の核

雨の駅前で再会する恋愛劇から始まり、異世界設定、叙述トリック、映画撮影現場、創作者たちのメタ視点へと階層がずれていく物語。

表面上はウィットのある破綻コメディだが、奥には「喪失をフィクションでどう扱うか」という主題を置く。

## ファイル

- `00_concept.md`: 作品の核、ルール、トーン
- `01_structure.md`: 章立てとオムニバス構成
- `02_characters.md`: 初期登場人物メモ（現在の正本は `10_役割別作業場/00_共通資料_登場人物深掘り/`）
- `03_opening_draft.md`: 現時点の本文サンプル
- `99_ideas.md`: 追加アイデア置き場

## チーム運用フォルダ

- `00_運用管理`: チーム全体のルール、作業手順
- `10_役割別作業場`: 原作、考証、脚本、編集などの担当別作業場。`00_共通資料_登場人物深掘り` は人物深掘り正本
- `20_校了品`: 合意済みの正本、作品設定の正本
- `30_引き継ぎ`: 担当間の依頼、質問、申し送り
- `40_レビュー`: 問題点、改稿方針、レビュー結果
- `50_出力物`: 外に出すための成果物
- `60_制作ラウンド`: 3,000字程度ごとの半自動制作単位
- `70_オーナー窓口`: ユーザーからの直接指示、相談、雑談、差し込み依頼
- `90_アーカイブ`: 没案、旧稿、試作
