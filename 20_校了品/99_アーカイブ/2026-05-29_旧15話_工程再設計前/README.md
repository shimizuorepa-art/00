# 旧15話アーカイブ

## 位置づけ

このフォルダは、2026-05-29の制作工程再設計前に校了扱いだったプロローグ5話と第1章10話を退避したもの。

旧稿は失敗稿ではなく、骨子、出来事、人物配置、使えそうな台詞、現場の手触りを残した試掘版として扱う。

## アーカイブ理由

- 各工程の役割を縛りすぎ、本文がチェックリストを通過した文章に寄った。
- 文体リライトに負荷が集まり、初稿側で人物を動かす力が弱くなった。
- 工程mdが「事故防止」中心になり、AIがのびのび本文を作る余地が狭くなった。
- キャラクター設定を章構成へ生かす仕組みが弱かった。

## 今後の使い方

- 骨子と出来事は参照してよい。
- 本文の言い回しは、そのまま正本へ戻さない。
- 新しい章骨子とキャラ会議を通して、必要な場面だけ再制作する。
- 旧稿と新版が矛盾した場合は、新版の章構成、完成稿、人物更新ログを優先する。
