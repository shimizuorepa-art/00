# 登場人物深掘り正本

このフォルダは、登場人物深掘りの正本として扱う。

単独の制作担当ではなく、01_原作から09_プロデューサーまでが毎回参照する共通資料である。
本文、章構成、脚本、レビュー、整合性確認で人物解釈に迷った場合は、まずこのフォルダを見る。

`02_characters.md`、`04_撮影現場主要人物.md`、`60_制作ラウンド` 配下の人物深掘りラウンドは、初期案または制作過程ログとして扱う。
矛盾した場合は、このフォルダを優先する。

主要人物を一人1フォルダで管理する。

意味がなさそうな設定も捨てずに置く。身長、足のサイズ、髪質、癖、初恋、性癖、家族、先祖、出身地、苦手な食べ物のような細部が、あとで台詞、所作、回想、ギャグ、伏線として使える可能性がある。

生々しい弱さ、醜さ、嫉妬、生活の圧、自己正当化、他者への嫌な感情も、この作品の人物を立てる重要素材として残す。
ただし本文で使うときは、露悪としてそのまま説明せず、行動、沈黙、段取り、台詞のずれへ変換する。

各人物には本名を付ける。役名だけで呼ばず、本人の人生と作中映画での役割を分けて管理する。

各人物に、生まれてから現在までの人生年表と、その人を象徴するエピソードを用意する。

## 正本ルール

- 人物深掘りの正本は、このフォルダ配下の `人物深掘り.md` と `00_相関図.md`。
- 初期メモやラウンド成果物と矛盾した場合は、このフォルダを優先する。
- 旧メモにだけある要素を採用したい場合は、このフォルダへ反映してから本文へ使う。
- 生々しい設定は削って丸めない。必要に応じて「本文ではまだ伏せる」「候補」「採用済み」を追記して管理する。
- サブエージェントへ発注するときは、視点人物、登場人物、強く言及される人物の該当ファイルを必ず読むものに入れる。
- 各人物ファイルの冒頭に `文体リライト用クイック参照` を置く。文体担当はまずそこだけを見る。
- 深層素材、人生年表、過去の傷、濃い関係性は、編集から「深層へ潜る」合図がある時だけ参照する。

## ラベル

- `本文表層`: 初登場から本文で出してよい薄味の核。
- `採用済み`: 本文・構成で使ってよい設定。
- `候補`: 面白いが、使う前に編集判断が必要な設定。
- `本文ではまだ伏せる`: 採用済みだが、開示時期を管理する設定。
- `深層素材`: 人物の奥行き用。説明台詞や地の文で直出しせず、所作、沈黙、段取り、台詞のずれへ変換する。
- `旧案`: 現行正本と矛盾する、または現時点では採用しない案。参照は可、本文投入は不可。

## フォルダ

- `00_相関図.md`: 撮影開始時点、進行で変わる関係、裏接点候補
- `01_悠人役`: なろうとした男
- `02_美咲役`: なった人
- `03_魔王役`: なってしまった人
- `04_監督`: 外側から映画を成立させようとする人
- `05_脚本`: 破綻に筋を通す人
- `06_原作者`: 無責任な想像を投げた人
- `07_AD`: 現場を止めない人

## 使い方

1. まず埋めやすいところから埋める。
2. 矛盾しても消さず、候補として残す。
3. 本名、人生年表、象徴エピソードは優先して埋める。
4. 本文で採用した設定は、このフォルダと整合性台帳へ反映する。
5. 面白いが未確定の設定は、このフォルダに寝かせる。
6. 文体担当に毎回見せたい特徴は、各人物の `文体リライト用クイック参照` に数行で置く。
