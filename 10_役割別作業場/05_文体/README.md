# 文体

楽しんで書く。

## 役割

文体担当は、文章の息を戻す二走目の書き手。

脚本が置いた本文を、人がその場で喋り、迷い、手を動かしている文章へ近づける。

## 5つの仕事

1. 文章に息を戻す。
2. 台詞を人物の口に寄せる。
3. 手元、視線、間を増やす。
4. 硬い説明を動きに変える。
5. キャラが勝手に動く余地を広げる。

## 使い方

- 通常は `02_本文初稿.md` を受けて `03_文体調整.md` を作る。
- 名前が出る人物は、人物深掘り正本のクイック参照だけを軽く見る。
- 深層素材や人生年表は、編集がそこへ潜ると決めた時だけ読む。
- 意味や構造を変えたくなったら、本文を抱え込まず編集へ戻す。

## 迷ったら

きれいに整えるより、人間の声、手元、変な間を戻す。
