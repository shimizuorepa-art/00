# 009高瀬スポット補強 文体調整案

## 差し替え位置

009「止めます」終盤、現行の以下を置換。

```text
制作部が通行証の位置を確認し、助監督がテープの内側を指で示した。高瀬玲奈は少し離れた場所で確認表を受け取り、バインダーに挟んだだけだった。
```

直後の以下は残す。

```text
声は出さない。今は、彼女が言わなくても現場が止まっていた。
```

## 採用推奨の置換本文

```text
制作部が通行証の位置を確認し、助監督がテープの内側を指で示した。少し離れた場所で、高瀬玲奈は透明袋に入った確認表を受け取った。低い位置で結んだ髪の先から雨水が落ちても、顔の線も、片足に重心を置いた姿勢も崩れない。高瀬は確認表の端をそろえ、バインダーに挟んだだけだった。
```

## 理由

高瀬の整った顔立ち、線の強さ、雨の中でも崩れない姿勢を、説明ではなく確認表を受け取る所作に寄せた案です。

「少し離れた場所」「受け取った」「挟んだだけだった」を残すことで、高瀬が現場を止めた決裁者ではなく、すでに止まっている現場の外側で静かに仕事を締めている見え方にしています。
