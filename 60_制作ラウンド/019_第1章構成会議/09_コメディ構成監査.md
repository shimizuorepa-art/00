# 09 コメディ構成監査: 第1章構成会議

## 監査結論

現在の第1章構成は、「馬車を引っ張りすぎない」「劇中劇を舞台装置にする」「主要人物全員に一度ずつスポットを当てる」という方針まで整理できている。方向性はよい。

ただし、章構成メモ内にまだ「業」「痛み」「怖さ」「代償」「消耗」といった重い語が残っており、このまま本文へ入ると第1章の入口として読者が身構えるおそれがある。第1章では、深刻な芯を消すのではなく、先に「この現場の人たち、面倒だけど好きだな」を作るのがよい。

コメディの軸は、大事件や異世界設定ではなく、真面目な現場確認のズレ、職能ごとの困り方、言葉の聞き間違い、仕事中の癖に置く。笑わせる対象は人物の無能さではなく、全員が真面目に仕事をしているせいで変なことになる現場そのものにする。

## 重く見えすぎる箇所

- 章の大目的、章末目標、人物運用で「痛み」「業」「怖さ」「代償」が並ぶ箇所は、第1章としては重く見える。芯としては残してよいが、本文上では「文句を言いながら手が動く」「確認事項が増える」「一度だけ黙る」程度の軽い現れに置き換えたい。
- 004の佐伯回は、引退気配を強く出すと初登場章から影が濃くなる。まずは「変な台詞の爆心地を自分で拾う人」「先に謝ってしまう人」として愛着を作り、去る気配は台本を閉じる手元や視線で薄く置く。
- 005の朝倉回は、消耗や怖さより「プロなので文句を言いつつ一回身体で試してしまう」おかしさを先に見せる。封筒や美咲の感情は、その動きが妙に成立してしまったあとに少しだけ沈ませる。
- 006の鳴海回は、原作者の後出し改変が強くなりすぎると鳴海が章の主役になる。楽しそうに火種を投げるが、ADや脚本の沈黙に一拍だけ気づく、くらいで人間味を残す。
- 008の撤収後は、後始末を重い余韻にしすぎない。濡れ物、車両、近隣対応、帰路確認など、残業の具体物で軽く見せるほうが現場コメディになる。
- 009から010は、本気化を強めすぎる危険がある。笑いが本気へ変わる瞬間は一回で十分。最後は「大変な映画」ではなく「この人たちが次も不満を言いながら動く」で締めたい。

## 軽めコメディに寄せる具体策

- 各話の入口は、世界観説明ではなく現場確認から入る。
  - 例: 「馬車は夢じゃなくて幅と重量です」
  - 例: 「魔王様、登録は一般エキストラで合ってますか」
  - 例: 「合成って言った人、素材の責任者も言ってください」
- 各人物に、読者が覚えやすい仕事中の癖を一つ置く。
  - 高瀬: 連絡しながら眉だけで会話する。
  - 久我: いい画を見るほど声が小さくなる。
  - 真鍋: 文句を言いながらメモの行数が増える。
  - 佐伯: 自分の台詞が爆心地になると先にスタッフへ謝る。
  - 朝倉: 無茶な芝居ほど一度身体で試してから文句を言う。
  - 鳴海: 思いついた瞬間は楽しそうだが、現場の沈黙で少しだけ現実に戻る。
  - 山野: 呼ばれるまでは店長、呼ばれた瞬間だけ魔王になる。
- 深刻な言葉が二段続いたら、現場の小動作で戻す。
  - 台本のページが雨でへたる。
  - 無線が別件を拾う。
  - 誰かの確認事項が一つ増える。
  - 山野が店内から普通に呼ばれる。
  - 高瀬が感情論を寸法と導線に戻す。
- 大ネタで笑わせようとしない。鳥車、魚、ドラゴン、勇者、魔王は一発の設定笑いではなく、人物の反応を引き出す札として使う。
- 誰かを悪者にして笑わない。鳴海も久我も真鍋も高瀬も、困った人ではあるが、それぞれ仕事をしている人として見せる。

## 各話の入口/出口コメディ案

| 話数 | 入口で使える軽いズレ | 出口で使える軽いズレ |
|---|---|---|
| 001 | 高瀬が「馬車」を情緒ではなく寸法、重量、導線、撤収時間で数え始める | 久我だけがまだモニターを見ていて、高瀬の確認事項が一つ増える |
| 002 | 「消す」「残す」がVFX用語、監督の未練、ADの実務で噛み合わない | 鳴海の「乗ってきてます」で、処理済みのはずの馬車が台詞として戻る |
| 003 | 真鍋が消す案を書こうとして、消さない案のほうが先に筋が通る | 「嫌だな」と言いながら、真鍋の台本余白だけが一番埋まっている |
| 004 | 佐伯が「言ったの俺です」と先に申告し、周囲が妙に助かる | 佐伯の真面目さが、笑いの火元を少しだけ芝居に変える |
| 005 | 朝倉が魔法の手の動きを試し、「これ特撮じゃないですよね」と確認する | 文句を言った本人の動きが一番使えそうになり、周囲が言い出せなくなる |
| 006 | 鳴海の改変案に、脚本とADが同時に別方向の現実問題を出す | 鳴海が少し楽しそうな顔をして、久我と真鍋が同時に黙る |
| 007 | 店内から「店長」と呼ばれ、名札と状況が二重に魔王を念押しする | 山野本人より、周囲の確認事項が先に魔王扱いへ進む |
| 008 | 撤収後なのに、馬車だけがまだ仕事を増やしている | 高瀬と佐伯が同じ馬車を別の意味で見ていると分かるが、会話は短く切る |
| 009 | 読み合わせ前に「馬車で来た理由」が妙に現実的に検討される | 役者が本気にした瞬間、真鍋だけが「今の、書いたの俺か」となる |
| 010 | 共有事項が増えすぎ、誰かが「今日は馬車だけで終わる日では」とこぼす | 魚やドラゴンより、全員が不満を言いながら次の仕事へ向く顔で締める |

## 劇中劇・馬車・魚・ドラゴンを主役化しない注意

- 馬車は001〜002で暫定処理する。003以降は「馬車をどうするか」ではなく、「残った馬車を見て、その人物がどう反応するか」へ焦点を移す。
- 魚、ドラゴン、勇者、魔王は「次のトラブル一覧」にしない。各人物の職能、癖、責任感を照らすための札にする。
- 魔王店長は設定説明にしない。山野が店長として普通に呼ばれ、名札と状況によって周囲が勝手に魔王扱いしてしまう現場コメディにする。
- 鳴海の改変案は、鳴海だけが面白がって勝つ形にしない。必ず脚本、AD、監督、役者の誰かが別方向の現実問題として受ける。
- 章末の引きは「次はドラゴンだ」ではなく、「次に何が出ても、この人たちは文句を言いながら仕事にする」へ置く。
- 大ネタを本文内で説明しすぎない。魚やドラゴンの詳細は、出すとしてもメモ、共有事項、読み合わせの端に留める。

## 章構成メモに反映すべき文言案

- 章のトーンに追加:
  「コメディは大ネタではなく、真面目な現場確認のズレで作る。人物の弱さは笑いのあとに短く置き、読者の愛着を先に作る。」
- 人物スポット方針に追加:
  「各話の主スポット人物には、欠落や過去説明より先に、仕事中の癖、言い方、反応のかわいげを一つ置く。」
- 10話割り方針に追加:
  「各話の入口と出口に、その人物らしい小さな現場コメディを一つ置く。馬車や異世界要素は、その人物を立たせるための小道具に留める。」
- 各話の切断ルールに追加:
  「深刻な事情は1話に1滴まで。重い言葉が続いたら、確認事項、濡れ物、無線、呼び込み、言い間違いなどで一度軽く戻す。」
- 入れすぎ注意に追加:
  「魚、ドラゴン、勇者、魔王は設定の面白さで押さず、現場側の反応で面白くする。章末の主役は新設定ではなく、次も動く人物たちに置く。」

## 優先反映順

1. 章全体の語尾から「業」「怖さ」「代償」を減らし、「癖」「手数」「困り方」「仕事にしてしまう」に置き換える。
2. 001〜010の各話に、入口コメディと出口コメディを一つずつ明記する。
3. 佐伯、朝倉、真鍋の重さは残しつつ、最初の印象を「笑える不器用さ」に寄せる。
4. 010の引きを、魚やドラゴンの強い予告ではなく、人物たちの次行動へ寄せる。
