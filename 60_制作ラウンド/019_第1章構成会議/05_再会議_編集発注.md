# 05 再会議 編集発注: 第1章方向性再確認

## 再会議の理由

前回の統合構成は、馬車問題を早めに処理し、主要人物にスポットを当てる方向へ修正した。ただし、その修正は補佐くん単独判断の比重が高かったため、正式に構成会議を開き直す。

## オーナー追加方針

- 劇中劇は舞台装置である。
- 馬車、鳥車、魔王、勇者、ドラゴン、雨粒の魚は、話を進め、登場人物を立たせる小道具であって、主役にしてはいけない。
- 第1章では、読者が主要登場人物の像を掴めることを優先する。
- 「馬車は残る」は、馬車そのものを引っ張る章ではなく、残ってしまった馬車をきっかけに各人物の視点、職能、痛み、欲、関係が見えてくる章にする。
- 主要登場人物全員に一度ずつスポットを当てる。

## 再判定してほしいこと

1. 第1章10話が、馬車/劇中劇をメインにしすぎていないか。
2. 主要7人のスポット順は妥当か。
3. 各話で、舞台装置が人物を立たせる方向に機能しているか。
4. 馬車問題をどこまで早く暫定処理すべきか。
5. 第1章の章末引きは、劇中劇の新ネタではなく人物の次の行動へつながっているか。
