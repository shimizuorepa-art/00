# 06 構成作家 舞台装置化再設計

## 前提

第1章は「馬車は残る」を扱う章ではあるが、馬車を主役にしない。

馬車、鳥車、かぼちゃの馬車、魔王店長、勇者設定、雨粒の魚、ドラゴンは、いずれも人物を立たせるための舞台装置である。読者に渡すべき中心体験は、「現場が変なものに振り回される」ではなく、「変なものを前にした時、その人物が何を守り、何を欲しがり、何を諦めきれていないかが見える」である。

したがって、第1章の10話構成は以下の優先順位で再整理する。

1. 主要7人を一度ずつ明確に立たせる。
2. 馬車問題は002までに暫定処理し、003以降は処理可否を主題にしない。
3. 劇中劇要素は、人物の職能、痛み、欲、関係を照らす役割に限定する。
4. 章末の引きは新設定の派手さではなく、人物が次に何を選ぶかへ接続する。

## 現在の10話構成の良い点

- 001〜002で高瀬と久我を立て、現場実務と演出欲の衝突から入る点は強い。読者が撮影現場パートへ入る足場になる。
- 馬車を「夢の絵」ではなく「幅と重量」「安全導線」「許可」「別撮り/合成」に分解する方針は、職業コメディとして有効。
- 003の真鍋、004の佐伯、005の朝倉、006の鳴海、007の山野という並びは、主要人物を順番に立てる意識がある。
- 009で「笑える破綻が役者の本気に変わる瞬間」を置く設計は、章の感情的到達点として妥当。
- 白い封筒、時計、異世界混線の真相を確定しない方針は守られている。
- 鳥車を採用せず、かぼちゃ案も保留する線引きは適切。

## 現在の10話構成の危ない点

- 章目的の文言がまだ「馬車問題を複合問題へ引き上げる」に寄りやすく、読者体験が「馬車会議を読んでいる」印象になる危険がある。
- 001〜003で馬車実務、久我の画、真鍋の脚本処理が続くため、会議説明が長く見える可能性がある。002で暫定処理を切り、003では明確に人物回へ移る必要がある。
- 006の鳴海回が強くなりすぎると、原作者が章の主役に見える。鳴海は「矛盾を投げる人」であって、全員を動かす神にしない。
- 010の引きが魚/ドラゴンの新メモ中心になると、再び劇中劇が主役化する。章末は「新ネタが来た」ではなく、「その新ネタを前に、各人物が逃げられなくなった」に寄せる。
- 山野回は魔王店長の設定説明に流れやすい。山野自身の「なってしまった怖さ」と、周囲の反応を主題にする。
- 高瀬と佐伯の裏接点、佐伯の引退、山野の抜擢理由などを出しすぎると、章の情報量が過密になる。第1章では匂いまで。

## 章方針: 劇中劇をメインにしない

第1章の主語を、各話ごとに以下のように固定する。

| 装置 | 主役にしないための扱い | 人物を立てる役割 |
|---|---|---|
| 馬車 | 002までに「今夜は動かさない。安全条件下の別撮り/合成前提で画には残す」へ暫定処理する | 高瀬の停止線、久我の画への執着、真鍋の書けてしまう業、佐伯の責任感を照らす |
| 鳥車 | 採用しない。回避された前例としてのみ扱う | 現場が「止められる無茶」と「止めきれない無茶」を区別する材料 |
| かぼちゃの馬車 | 保留。真鍋が書けてしまう誘惑として置く | 破綻に筋を通せてしまう真鍋の怖さを示す |
| 魔王店長 | 世界観説明をしない。山野光の役、画面上の残り方として扱う | 山野の「なってしまった人」性、久我の発見欲、佐伯/朝倉の揺れを照らす |
| 勇者設定 | 章内でバトル化しない。佐伯が抱える悠人像のズレとして扱う | 佐伯の「最後に見られたい」欲と、朝倉の相手役としての苛立ちを出す |
| 雨粒の魚 | 010の引きまで温存。絵として説明しない | 鳴海の嘘が現場に渡った後、真鍋や役者がどう責任を持つかへ送る |
| ドラゴン | 010の引きまで温存。VFX説明や実体化はしない | 久我の「また捨てられない画が来る」予感、高瀬の次の停止線を作る |

## 馬車問題の暫定処理

馬車問題は002までに暫定処理する。

- 001: 高瀬が「今夜この場では動かさない」と一時停止を作る。
- 002: 久我が安全条件を飲み、「動かさないが画からは消さない」を受ける。処理方針は、安全確認後の別撮り/合成/小道具車両素材の組み合わせに仮固定する。
- 003以降: 「馬車をどうするか」へ戻らない。「残ることになった馬車が、誰の何を照らすか」へ移る。

ここでの暫定処理は、馬車問題の消滅ではない。実務上の可否を止め、人物ドラマの火種として残す処理である。

008〜010は単独スポットではなく、人物同士の接続回にする。

- 010: 全員。劇中劇の新ネタではなく、各人物が次に引き受ける行動を見せて章を閉じる。

## まだ出しすぎない情報

- 佐伯の引退予定の確定。第1章では気配まで。
- 高瀬と佐伯の同郷、時計伝説、引退共有の全面開示。使う場合も「知っているかもしれない」反応まで。
- 山野がなぜ画面に残るのかの理由づけ。第1章では現象と周囲の反応まで。
- 鳴海の本心。現場を笑っているが、笑いきれていない程度まで。
- 白い封筒、駅前時計、悠人の不在理由、異世界混線の原理。
- 魚、ドラゴン、勇者/魔王対立の実体化。第1章では次章への圧に留める。

## 結論

現行案の骨格は採用できる。ただし章の主語を「馬車問題」から「馬車が照らす7人」へ明確に移す必要がある。

最重要修正は、馬車の実務可否を002までに暫定処理すること。003以降は、馬車や劇中劇の新設定を説明するのではなく、それぞれの人物が何を引き受け、何を見られたくなく、何をまだ諦めていないかを見せる回として組む。

章末の引きも、魚やドラゴンの派手さで引っ張らない。鳴海の新メモは出してよいが、最後に残すべき読後感は「また変な設定が来た」ではなく、「この人たちは次の撮影で、もう逃げられない」である。
