# 10 読者愛着レビュー: 第1章構成会議

## レビュー結論

第1章は「何が起きるか」より「誰がどう受けるか」で読ませる章にしたほうが強い。現状の10話構成は人物スポットの順番が整理されているので、各話に読者が好きになる一手を明示すれば、軽めコメディ章として機能する。

読者が愛着を持つ入口は、立派な信念よりも先に、仕事の癖、困ったときの言い方、他人への小さな配慮、ツッコミ損ねた表情でよい。

## 人物別の愛着ポイント

- 高瀬玲奈: 口はきついが、誰より先に逃げ道と安全を作る。怒るためでなく続けるために止める人。
- 久我修一: 良い画に負けるが、現場の人間を無視しているわけではない。止められた時の間にかわいげを作る。
- 真鍋由規: 文句が多いのに、誰より整えてしまう。嫌そうな顔と手の速さの差が魅力。
- 佐伯蓮: 爆心地なのに人のせいにしない。自分の台詞を引き受ける真面目さを、少し照れた笑いで見せる。
- 朝倉澪: もっとも文句を言う権利があるのに、結局いちばん芝居で返してしまう。プロの反射が愛着になる。
- 鳴海千景: 愉快犯に見えるが、現場の沈黙に一瞬だけ気づく。悪魔的な楽しさと、完全には無神経でいられない揺れを両方置く。
- 山野光: 本人は普通に店長業をしているのに、周囲が勝手に魔王として扱い始める。そのズレが愛らしい。

## 各話で読者に渡す「好きになる一手」

| 話数 | 好きになる一手 |
|---|---|
| 001 | 高瀬が誰かを責める前に、まず一般客とスタッフの逃げ道を確保する |
| 002 | 久我が無茶を言いかけて、高瀬の条件をちゃんと飲む |
| 003 | 真鍋が「無理」と言いながら、誰より早く成立案を出す |
| 004 | 佐伯が台詞の爆発を自分のものとして引き受け、周囲を楽にする |
| 005 | 朝倉が文句を言いつつ、白い封筒を持つ手だけは芝居から逃がさない |
| 006 | 鳴海が現場を困らせるが、ADの沈黙に一拍だけ反応する |
| 007 | 山野が魔王扱いより先に、店内のバイトを気にする |
| 008 | 高瀬の実務と佐伯の役者としての気遣いが一瞬だけ重なる |
| 009 | 真鍋、朝倉、佐伯が同じ台詞を別々に本気にしてしまう |
| 010 | 全員が不満を言いながらも、次の仕事へ身体が向いている |

## 注意点

- 愛着を作る前に引退、喪失、嘘の怖さを前面に出しすぎない。
- 鳴海は好き勝手に暴れるだけだと読者の反感が先に立つ。現場の反応を見る一瞬を必ず入れる。
- 久我は無責任に見えやすい。高瀬の条件を飲む、役者の反応を見て黙るなど、現場を見ている証拠を置く。
- 朝倉は有能すぎると距離が出る。特撮未経験への戸惑い、手の動きへの照れを残す。
- 山野は魔王設定の説明より、普通の人としての反応を先に出す。

## 章構成メモへ反映すべき文言案

- 「人物紹介は、欠落や過去ではなく、仕事中のかわいげから入る。」
- 「各話の主スポット人物には、読者がその人を少し好きになる行動を一つ置く。」
- 「章末では破綻の大きさより、全員が不満を言いながらも次の仕事へ向く感じを残す。」
