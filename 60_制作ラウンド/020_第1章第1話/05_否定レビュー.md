# 否定レビュー: 第1章第1話「馬車は幅と重量」

## 判定

条件付き合格。

場面目的はおおむね達成している。馬車を「夢」から「幅と重量」に落とし、高瀬玲奈を読者の足場にする流れは成立している。高瀬が怒鳴る前に導線を空ける点も、人物深掘りの「現場を止めずに、誰も壊さずに、今日の撮影を終わらせたい」に合う。

ただし、否定レビューとしては小リテイクを推奨する。主なリスクは、終盤で高瀬の一時停止が単独決裁に見えかけること、モニター上の馬車が一瞬だけ美化されすぎて主役化しかけること、無線コメディがやや畳みかけ型になっていること。この三点を軽く締めれば、本文の狙いはより安定する。

## 優先リテイク

### P1. 高瀬の権限をもう一段だけ限定する

対象: `03_脚本ドラフト_文体調整.md` 83行目、129〜133行目付近。

問題点:
「馬はなしです。鳥もなしです」「鳥車も馬も増やしません」は場面上は自然だが、単独決裁者ではない高瀬が最終採否まで切っているようにも読める。直前に「上に戻します」「確認が済むまで一回止めます」があるため大事故ではないが、発注上の重点が「高瀬を単独決裁者にしない」なので、ここは念押しした方が安全。

修正方向:
高瀬の判断を「今夜この場での一時停止」「確認済みまでの停止」「上位者・関係部署へ戻すための停止」に限定する。たとえば、最後の宣言に「この場では」「確認が戻るまで」を重ねる、または制作・助監督部側の復唱を一行入れて、高瀬の線引きが組織内の確認導線に乗ったことを見せる。

最低限の直し:
「鳥車も馬も増やしません」を「確認が戻るまで、鳥車も馬もこの場では増やしません」程度に寄せる。

### P1. 終盤の馬車描写を高瀬の仕事へ戻す

対象: `03_脚本ドラフト_文体調整.md` 121〜127行目付近。

問題点:
「駅前の光と雨を拾っていた」「嘘みたいにきれいに回り込んでいる」は、次話へ渡す引きとして機能している。一方で、この話の主役は馬車ではなく高瀬なので、ここだけ馬車の画が少し強くなりすぎる。久我が画から離れられない理由を置くには必要だが、読者の感情が馬車の美しさへ寄りすぎると、高瀬回の終盤が「馬車が魅力的だった話」へ傾く。

修正方向:
美しさを残す場合も、高瀬にはそれが「反射確認」「既撮り素材」「撤収後の手戻り」として見えるようにする。馬車の絵面を褒め切らず、見た瞬間に高瀬の確認事項へ変換される流れを強める。

最低限の直し:
「嘘みたいにきれい」を少し抑え、「きれいだから困る」「映り込みとして残ってしまう」方向へ寄せる。

### P2. 無線コメディの連打を一拍減らす

対象: `03_脚本ドラフト_文体調整.md` 79〜95行目付近。

問題点:
「馬はなし」「鳥もなし」「かぼちゃは」「野菜の話」の流れは軽く読めるが、三段オチとして整いすぎている。第1章の方針は大ネタではなく真顔の確認事項のズレなので、この密度だと少しだけ内輪の掛け合いに寄る。特に無線上で「かぼちゃ」が拾われると、現場全体がボケを回しているように見え、安全確認の実務感が薄くなる。

修正方向:
笑いを残すなら、無線は「馬」または「鳥」までに絞り、「かぼちゃ」は鳴海の口頭火種として短く処理する方が現場の真面目さを保てる。

最低限の直し:
91〜95行目を削るか、無線ではなく鳴海の小声にして、高瀬が小さく遮る程度にする。

### P2. 冒頭の高瀬がツッコミ役に見えるリスクを下げる

対象: `03_脚本ドラフト_文体調整.md` 1〜15行目付近。

問題点:
冒頭第一声が「書けてしまう、じゃないです」なので、高瀬の初手がツッコミに見える。その後すぐに現場全体を見て導線確保へ移るため回復しているが、発注の重点は「高瀬をツッコミ役だけにしない」なので、初印象だけ少し危ない。

修正方向:
第一声は残してよい。ただし、声を出す前に「現場を見た」「人が詰まっていた」を先に置く、または第一声の直後に無線指示まで滑らせると、ツッコミではなく実務反射として読める。

最低限の直し:
1行目のあと、もう少し早く「誰のせいかより先に導線」を接続する。

### P3. バス導線を言葉だけで終わらせない

対象: `03_脚本ドラフト_文体調整.md` 9行目、63行目、101行目付近。

問題点:
「バス停の白線」「バス導線」は出ているが、平日夜の駅前ロータリーとしては、バスそのものの圧がやや薄い。考証上、馬車を動かせない理由の柱の一つがバス導線なので、もう一つだけ具体物があると説得力が増す。

修正方向:
バスの行先表示、発車待ち、車寄せ、エアブレーキ音、終バス前の列などを一つだけ入れる。増やしすぎると説明になるので、一行でよい。

### P3. 「ホースの内側」が少し曖昧

対象: `03_脚本ドラフト_文体調整.md` 15行目。

問題点:
「スタッフもホースの内側に残らない」は意味は取れるが、現場の位置関係として少し曖昧。ホースが輪になっているのか、導線の内側なのか、読者が一瞬考える。

修正方向:
「ホースの外へ」「ホースを跨がない位置へ」「ホースより駅側へ」など、退避方向が見える言い方にする。

### P3. 「書類に残ると嫌」は安全確認として少し危うい

対象: `03_脚本ドラフト_文体調整.md` 87行目。

問題点:
「復唱しなくていいです。書類に残ると嫌なので」は笑えるが、事故・許可・安全の文脈では「記録に残したくない」ようにも読める。高瀬は隠す人ではなく、確認導線を作る人なので、ここだけ少し人格と職能がぶれる。

修正方向:
「確認項目が増えるので」「未確認の名詞を増やさないでください」などに寄せると、高瀬の実務者らしさを損なわずに笑いを残せる。

## チェック詳細

### 場面目的

大筋で外れていない。プロローグ第5話直後から始まり、前話の「かぼちゃの馬車」「書けてしまう」を受けて、高瀬の確認順へ移っている。終了点も「今夜この場では動かさない」「確認事項が一つ増える」で、構成メモの切りどころに合う。

弱点は、コメディが気持ちよく回るぶん、80〜95行目付近だけ目的が「安全確認」から「無線ボケの処理」へ少し寄ること。削るならここ。

### 高瀬玲奈

人物深掘りとの整合は高い。声が通る、怒鳴らず動かす、ペンを耳に挟んだまま探す、自分の仕事が増える方を先に見る、という要素はよく入っている。

問題は権限の見え方。43行目と129行目の「上に戻します」は良いが、83行目と133行目の断定が強いため、読者によっては高瀬が採否まで決めたように見える。高瀬は「止める人」ではなく「現場を続けるために確認時間を作る人」なので、停止の根拠と範囲を最後まで薄めないこと。

### コメディ

全体としては過剰ではない。笑いの対象が無能な人ではなく、真面目に確認した結果変な言葉が生まれる現場になっている点は良い。

ただし、馬、鳥、かぼちゃ、野菜が続く箇所は少し整理されたコントに近い。第1話は高瀬の足場作りが主目的なので、笑いは「馬車は幅と重量です」「寸法をください」「雨案件です」あたりで十分に機能している。追加のボケを残すなら一つまで。

### 劇中劇・馬車の主役化

劇中劇の設定説明には流れていない。鳥車、かぼちゃの馬車、本物の馬も採用していない。山野の「魔王」名札も、コンビニ前の生活圏に留まっていて主役化していない。

唯一、終盤のモニター内の馬車だけが少し魅力的に描かれすぎる。次話の久我回へ渡す引きとして必要だが、この話では高瀬の疲労と確認事項の増加が主役なので、馬車の画の美しさは「困る美しさ」に留めたい。

### 情報管理

早出し事故は見当たらない。悠人の不在理由、白い封筒と駅前時計の真相、異世界混線の原理、撮影現場パートの階層、佐伯の引退、高瀬と佐伯の裏接点、鳴海の本心、山野が画面に残る理由づけは出ていない。

佐伯と朝倉の77行目は顔見せとして許容範囲。ただし「何か言えば自分たちの出番が増えると分かっている顔」は少し脚本側の都合が透ける。高瀬視点寄りにするなら、「自分たちが動くと導線が増えると分かっている」程度の現場目線へ寄せてもよい。

### 土地・時間・雨・駅前ロケ・安全導線

東京西部の郊外駅前ロケとして大きな不自然さはない。雨、散水ホース、濡れた白線、作業灯、発電機、コンビニ、一般客の抜け道がそろっており、完全封鎖ではなく生活圏を借りて撮っている感じも出ている。

強化するならバス導線。現状でも言葉としてはあるが、馬車を動かせない理由の柱なので、バスの光、音、列、発車待ちのどれか一つを入れると安全判断の説得力が上がる。

## 残してよい点

- 高瀬が誰かを責める前に導線を空ける流れ。
- 若いスタッフを叱るのではなく役目へ戻す処理。
- 「馬車は夢じゃなくて幅と重量です」から寸法確認へ移る構造。
- 久我を無責任な敵ではなく、良い画に声が小さくなる人として置く描写。
- 真鍋の「書けてしまう」を便利な解決でなく、湿気と余白の小ネタに留めている点。
- ラストの「既撮りの馬車、モニター上の映り込み確認」は、次話への引きとして有効。

## 総合

大規模リテイクは不要。第1章第1話としての骨格は立っている。

必要なのは、権限、主役、笑いの密度の微調整。高瀬が「決めた」のではなく「確認が済むまで止めた」、馬車が「美しい」のではなく「美しいから確認事項が増えた」、無線が「ボケを拾った」のではなく「真面目な復唱が少しずれた」と読めるように寄せれば、この話の目的はより明確になる。
