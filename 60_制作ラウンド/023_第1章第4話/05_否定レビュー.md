# 05 否定レビュー: 第1章第4話「言ったのは俺です」

## 総評

文体調整稿は、発注の主要条件はおおむね守れている。馬車の実務再会議には戻っておらず、「馬車で来た」を佐伯本人の台詞と身体へ戻す回として成立している。制作メモの混入も見当たらない。字数も約2,700字で上限内。

ただし、否定レビューとして見ると、完成稿化前に削るべきリスクは残る。いちばん大きいのは、佐伯回であるはずなのに、中盤の真鍋・高瀬の応酬が強く、前話の真鍋回の延長に少し見えること。次に、「未送付」「役者確認前」の職能線が明確な反面、反復が多く、読者によっては現場コメディではなく説明確認の往復に見える。佐伯の内面は自己分析台詞にはなっていないが、朝倉の一文がやや代弁に近い。

全体としては大幅リテイク不要。ただし、統合稿では会話量の圧縮と視点重心の再調整が必要。

## リスク/問題点（優先度付き）

### P1 佐伯回なのに、真鍋と高瀬の掛け合いが中盤を取りすぎる

「三段階で逃げ道」「導線」「戻したくない」「戻すかどうかも確認します」から、「責任感です」「駄目じゃないから困るんです」までの応酬は面白いが、かなり真鍋が主役に戻る。第3話で真鍋の言葉が強かった直後なので、ここでまた真鍋の脚本家としての怖さ・面白さを押すと、佐伯の「自分に戻ってきた」感より、真鍋の「書いたものが戻ってくる」感が勝つ。

特に「駄目にできるものは、まだ楽です」は真鍋の深い本音に寄っていて、佐伯の回としては一段いい台詞すぎる。真鍋を下げる方針に対して、ここだけ真鍋の人物深掘りが進みすぎる。

### P1 高瀬が確認役ではなく、判断権者に見える箇所がある

高瀬の「未送付です」「仮です」「役者確認前です」は職能線として有効。ただし「戻すかどうかも確認します」は、彼女が戻す/戻さないを決める側に見えやすい。人物深掘りでは、高瀬は単独決裁者ではなく、確認導線を作る人。ここは「戻すかどうかも確認に上げます」「戻す前に止めます」くらいにしないと、助監督部の線を越えて見える。

### P2 「未送付」「役者確認前」の説明は明確だが、やや重い

冒頭の紙面、スタッフ会話、高瀬の割り込み、整理台詞、真鍋の念押しで、同じ状態が何度も確認される。読者に職能線は伝わるが、情報としては過剰気味。現場のズレで笑うより、「これは未送付です」というルール説明を何度も聞かされる印象になる恐れがある。

最低限、後半の「まだ役者側に戻してません。だから、今のこれは正式な話ではありません」は、前半ですでに読めている。残すなら、佐伯に刺さる言葉として短くする方がよい。

### P2 朝倉の一文が、少し佐伯を解剖している

「謝っている。けれど、全部消してほしい人の顔ではない」は短く抑えているが、朝倉が佐伯の内面をかなり正確に言語化している。第5話への渡しとしては有効だが、今回の指示は「一拍だけ気づく」「言葉で解剖しない」。ここは読者に説明を渡しすぎる寸前。

「全部消してほしい人の顔ではない」は、佐伯の欲をかなり直接名指ししている。朝倉の視線と、膝上の指の形だけで止めるか、「消したい、だけではない」程度に曖昧化した方が次話への余白が残る。

### P2 情報漏れの自然さが、少しだけ都合よく見える

佐伯の近くでスタッフが「佐伯さんの台詞、残す場合って」「でも消す場合も」と言う導入は機能している。ただ、役者本人が袖口を直されている真横でその話が出るため、現場の不用意さが強く見える。考証メモでは「スタッフの断片」「小声」「佐伯さん呼びます？」程度なら自然とされているが、現稿は断片が都合よく核心に近い。

「無線越しに聞こえた」「通りすがりのスタッフが高瀬に確認した」「佐伯が聞き取ったのは語尾だけ」など、漏れ方をもう少し雑音に埋めると事故感が自然になる。

### P2 ラストは良いが、直前が少し覚悟の台詞へ舗装されすぎている

「正式じゃないものを、正式に受け取らないでください」からの「言ったのは俺です」は、構造としてかなりきれい。きれいすぎる分、佐伯が完全に覚悟したように読める危険がある。発注上は「完全な覚悟や感動の解決にはしない」なので、現場の呼び込みと雨音で冷ましている点はよいが、真鍋の制止が強すぎるため、佐伯の返答が決め台詞化しやすい。

直前の真鍋の台詞を少し軽くする、または佐伯の言い方に迷いを残す描写を一つ足すと、解決ではなく危うさとして読める。

### P3 久我は許容範囲だが、登場の切れ味がやや強い

「それは見えていました」で入るため、久我が佐伯の芝居を評価・判定する人として強く出る。その後の「半歩。袖口。見てから言ったところ」は非常に良いが、導入の一言だけが監督の目線として強い。

久我を「画として拾うだけ」に留めるなら、「そこ、残ってる」系の事実提示から入った方が、佐伯の回を邪魔しない。

### P3 白い封筒への触れ方は許容だが、やや注意

「白い封筒の小道具はもう手元にはない。けれど、指だけが何かの角を押さえる形で膝の上に残っている」は、前話・次話への接続として効いている。一方で、隠す情報に「封筒の真相」があるため、読者の注意を封筒へ戻しすぎるリスクもある。今回は佐伯回なので、朝倉の身体感覚への渡しに留め、封筒そのものの意味を匂わせすぎない方が安全。

### P3 文体上の過剰演出が数箇所ある

「会話が一度、作業灯ごと止まった」「作業灯の白が、湿った紙の角で小さく折れる」など、文体としては綺麗だが、現場コメディ寄りの軽さから一瞬だけ叙情へ寄る。多用はされていないため事故ではないが、第1章前半の軽さを守るなら一部は削ってよい。

## 最低限直すべき点

1. 真鍋・高瀬の中盤応酬を2〜3往復削り、佐伯の所作と反応へ紙幅を戻す。
2. 高瀬の「戻すかどうかも確認します」を、単独判断に見えない言い方へ修正する。
3. 朝倉の「全部消してほしい人の顔ではない」を少し曖昧にし、佐伯心理の説明役にしすぎない。
4. 佐伯がスタッフ断片を聞く箇所を、もう少し雑音・移動・無線に紛れた漏れ方にする。
5. ラスト直前の真鍋の制止を少し弱め、「言ったのは俺です」が完全な覚悟台詞に見えないよう冷ます。

## リテイク要否

大幅リテイクは不要。

ただし、軽微修正ではなく「中盤の圧縮」は必要。現稿のままでも読めるが、第3話と続けて読むと真鍋の存在感が強く、佐伯のスポット回としての輪郭が少し薄くなる。完成稿前に、真鍋の良い台詞を惜しんで削る判断がいる。

## 編集統合への申し送り

- 「馬車問題が主役化しすぎる」事故は起きていない。馬車そのものではなく、佐伯の台詞と身体に戻っている。
- 佐伯は自己分析しすぎていない。むしろ成功している。ただし朝倉が一度だけ代弁しかける。
- 高瀬はおおむね確認役だが、一部の言い回しで決定権者に見える。統合時に職能線を守ること。
- 真鍋は面白いが、少し出すぎ。第3話からの連続感を考えると、ここは厳しめに削る。
- 久我は許容範囲。評価論や演出論には踏み込んでいないが、登場第一声は弱めてもよい。
- 雨、駅前、待機テント、衣装の応急処置、発電機、通行確保は考証上おおむね自然。問題があるとすれば、情報漏れの都合よさ。
- 隠す情報の重大な開示はない。封筒への視線だけ注意。
- 制作メモ等の混入は見当たらない。
