# 変化した人間関係

## 佐伯蓮 と 山野光

開始時点では、予定された俳優と予定外に画面へ選ばれた人だった。

第1章終了時点では、佐伯は山野を助けるだけの先輩でも、嫉妬するだけの俳優でもなくなった。山野が名前で呼ばれる瞬間を、助けず、急かさず、見逃さない人になっている。

山野はまだ佐伯の嫉妬や痛みを理解していない。ただ、佐伯が自分を見ていることは受け取れる距離に来た。

## 朝倉澪 と 山野光

開始時点では、プロの俳優と急に抜擢された素人寄りの出演者だった。

第1章では、朝倉が山野を説明で導かず、立ち位置と視線で関わった。山野は朝倉を怖い人として見ながらも、朝倉の短い指示で自分の居場所を少し直せた。

第2章では、朝倉が山野を守るのではなく、仕事として近づく関係が使える。優しさだけではなく、苛立ち、警戒、同じ場面を背負う責任が混ざる。

## 佐伯蓮 と 朝倉澪

第1章では直接の深い会話はまだ少ない。

ただし、佐伯の半拍の遅れ、朝倉の「できます」、三行の読み合わせで、互いに役者として相手の弱さや強さを感じる準備ができた。朝倉は佐伯の去る気配に気づきやすく、佐伯は朝倉の強さに甘えやすい。

第2章では、二人の絡みは恋愛的に進めるより、芝居上の距離と本人同士のズレを先に出す。

## 高瀬玲奈 と 山野光

高瀬は山野を直接救う人ではなく、登録確認や導線確認の一部として背景へ下がった。山野が本人として止まるきっかけは、高瀬一人ではなく衣装部、制作部、助監督、久我の判断へ分散した。

ただし、高瀬は山野のケア係ではない。第2章では、必要な手続きと安全確認をする人として置き、山野の心境は本人や役者同士の場面へ渡す。

## 高瀬玲奈 と 佐伯蓮

第1章では、佐伯と高瀬の過去接続はまだ表に出さない。佐伯は高瀬に止められるのではなく、山野の立ち位置を奪わないために自分で止まった。

高瀬が佐伯をどう呼ぶか、佐伯がそれをどう受け取るかは、次章以降の火種として残す。

## 真鍋由規 と 鳴海千景

第1章で、鳴海の軽い火種が真鍋の三行へ変換された。

鳴海は真鍋をからかうだけではなく、自分の面白がりが彼の仕事を増やすことに気づいた。真鍋は怒りながらも、鳴海の破綻から使える核を拾ってしまう。

第2章では、この二人の会話を増やすと、原作通りにしない条件の責任が軽口の中から出せる。

## 久我修一 と 山野光

久我は山野を説明で持ち上げず、画面に残った半歩を見た。

山野の理解や成長ではなく、本人が分かっていないまま画面に残るところを評価している。第2章では、久我が山野へ短い演出を投げる場面が使えるが、山野を監督の発見物にしすぎない。

## 鳴海千景 と 役者陣

鳴海は、佐伯、朝倉、山野を「原作改変の面白い結果」ではなく、それぞれの身体で受ける人として見始めた。

まだ責任を引き受けたわけではない。第2章では、鳴海が役者へ近づきたいが、言葉が発注や感想になってしまうズレを出すとよい。
