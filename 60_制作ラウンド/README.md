# 制作ラウンド

1回あたり本文3,000字程度までを扱う制作単位。

2026-05-29以降は、新制作工程を優先する。

## 通常回の標準ファイル

- `00_発注カード.md`
- `01_設計メモ.md`
- `02_本文初稿.md`
- `03_文体調整.md`
- `04_編集確認.md`
- `05_完成稿.md`

## 章単位の標準ファイル

- `キャラ会議.md`
- `章構成メモ.md`
- `章末レビュー.md`
- `章末人物更新.md`

## 追加ファイル

重要回、設定追加回、迷った回だけ、考証、演出、読者視点、整合性管理、プロデューサー、肯定レビュー、否定レビューを追加する。
