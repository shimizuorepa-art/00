# 02 考証メモ: 第1章第3話「書けてしまう道」

## 1. 考証対象の抜き出し

- 第2話直後の継続。雨の駅前ロータリー脇、モニターベース周辺、または数歩外れた待機位置から動かさない。
- 焦点は「馬車を今夜動かすか」ではなく、「既撮りの画に残すなら脚本がどう引き受けるか」。
- 真鍋の台本余白に残す項目は「消す場合」「残す場合」「どっちも台詞が増える」「馬車を残す理由」。
- 確認対象は、佐伯の「馬車で来た」系の台詞、朝倉の視線、沈黙、白い封筒を持つ手、モニター上の馬車の反射。
- 高瀬は決定権者ではない。確認線を引き、助監督部・制作部上位者・各部署へ戻す人。
- 真鍋は解説者ではなく、嫌がりながら欄、矢印、仮メモを整えてしまう脚本家。
- 仮理由「馬車は迎えではない。帰り道が、もう普通ではない証拠」は、真相ではなく脚本上の暫定メモとして扱う。

## 2. 土地・時間・天候・ロケ現場の確認

- 舞台は東京西部、日野市付近をモデルにした架空の郊外駅前。実在駅名、実在バス停名、実在店舗名は出さない。
- 夜の雨。春先なので、濡れた衣装は冷え、紙はふやけ、ケーブルや床養生の足元確認が増える。寒さは真冬ほどではなく、湿り気と撤収疲れを優先。
- 明かりは駅前照明、コンビニの白い光、モニターの青白さ、発電機まわりの作業灯。派手な都市夜景ではなく、郊外駅前の明るい場所だけ浮く感じが合う。
- 終バス前後のロータリーとして、バスの鼻先、一般車の一時停車、タクシー列、店前の自動ドア、歩行者導線を塞げない。終電・終バスの具体時刻は断定しない。
- 散水は止まりかけ。ホース巻き取り音、濡れたコード、滑る養生、レインカバー付きの機材、濡れた台本、曇るモニター、重く垂れた遮光布を使える。
- 近隣対応は「騒音を出さない」「発電機の向き」「作業灯を住宅側へ向けない」「店前導線を空ける」程度で十分。道路使用や許可の詳細説明へ戻さない。
- 馬車の安全、移動、許可、VFX素材、クリーンプレートの再会議は背景に退ける。本文では「戻す先は増えたが、今ここで決めない」と処理する。

## 3. 映像制作/脚本確認の職能メモ

- 現場での「脚本確認」は、決定稿の執筆ではなく、既撮り素材と今後の撮影に矛盾が出ないよう仮整理する作業として置く。
- 高瀬の言葉は「決定ではありません」「確認に戻します」「現時点では仮メモです」「上位者と各部署へ戻します」が自然。
- 高瀬に「採用します」「これで行きます」と単独決定させない。言えるのは「その前提で確認します」「消す場合も残す場合も確認します」まで。
- 真鍋の言葉は「今のは台詞ではありません」「確認であって執筆ではないですよね」「口が勝手に下書きしました」など、逃げと職能が同時に見える方向。
- 監督の久我は画を残したい圧として短く使う。「そこにあると、帰れない感じがする」程度でよい。長い美学説明は第3話の中心を奪う。
- 鳴海は火種だけ置く。「来た、より、帰れない、ですかね」程度。正解を知っている人、真相を語る人にはしない。
- 佐伯と朝倉は今回は本格的に読ませない。二人の台詞・目線・沈黙が脚本処理の問題として話題に出る程度で、第4話以降へ渡す。
- 現場の台詞差し替えは、正式な改稿ではなく「仮メモ」「差し替え候補」「確認用の一行」と呼ぶのが安全。

## 4. 本文に使える具体描写

- 濡れた台本のページが貼りつき、閉じようとすると「馬車を残す理由」だけが見える。
- ペン先がふやけた紙に沈み、真鍋の字だけ少し太る。
- 高瀬の確認表の端に「脚本確認」が足され、雨で罫線がにじむ。
- モニターの端で、コンビニ光を拾った車輪の縁だけが白く残る。
- 作業灯が少し振れて、路面の反射の中の馬車が一瞬薄くなる。
- バスの行先表示が雨粒で読みにくく、その奥に馬車の影が残る。
- 店の自動ドアが開くたびに白い光が伸び、濡れた衣装の袖口だけが明るくなる。
- 佐伯の袖口から水が落ちる。台詞を消す場合、その一拍を別の動きで受け直す必要がある。
- 朝倉が白い封筒の角を親指で押さえる。真相は出さず、「その手が止まった理由を、消すなら別に作る」と真鍋が嫌がる。
- 「消す場合」には、佐伯の台詞差し替え、朝倉の視線の逃げ場、沈黙の理由、白い封筒を触る間の処理が増える。
- 「残す場合」には、馬車がただの映り込みでなく、二人の帰り道を変える画として働き始める。
- 真鍋が「迎えに来たんじゃない」と否定した直後、余白に「迎えではない」と書いてしまっている。
- 高瀬が「仮メモですね」と言い、真鍋が「今のは事故です」と返す。
- ラストの一行は「馬車は迎えではない。帰り道が、もう普通ではない証拠」。横に小さく「仮」と丸で囲むと、最終解決ではないことが出る。

## 5. 違和感・避けるべき処理

- 馬車を再び動かす、安全確認をやり直す、道路使用やVFX素材の詳細会議へ戻る処理は避ける。第2話で進めた焦点が後退する。
- 実在駅名や具体の終電時刻を出して、現実の路線考証に縛られすぎない。架空郊外駅前の生活感で十分。
- 高瀬を現場の最終決定者にしない。単独で採用・不採用・撮影続行を決めると職能が強すぎる。
- 真鍋に状況を全部説明させない。説明は余白の小見出し、矢印、口から漏れた一言で見せる。
- 鳴海に真相めいた正解を言わせない。鳴海は笑いきれない火種まで。
- 久我の美学語りを長くしない。今回は真鍋回であり、久我はモニターを見続ける圧に留める。
- 佐伯の引退確定、高瀬との同郷・学校時計伝説、共有過去を出さない。
- 白い封筒と駅前時計の真相、悠人の不在理由、異世界混線の原理、撮影現場パートの階層を説明しない。
- 山野が画面に残る理由をここで扱わない。山野はコンビニ側の光、自動ドア、背景の気配程度に退ける。
- 仮理由を「正解」として周囲が感動する締めにしない。書けたせいで次の問題が増えた温度で切る。

## 6. 次工程への指示

- 冒頭は第2話ラストの余白から直結させる。場所移動や時間経過を大きく入れない。
- 本文の軸は、濡れた台本、モニター上の馬車、高瀬の確認表の三点で組む。
- 「消す場合」「残す場合」の両方に、台詞・目線・沈黙・封筒を触る間の負荷を短く置く。
- 真鍋の好感は、文句を言いながら佐伯と朝倉が困らない形へ整理してしまう行動で出す。
- 高瀬は「決定ではない」と線を引き続ける。真鍋の逃げを事務語へ変換する役にする。
- ラストは仮理由を書いた瞬間で切る。佐伯・朝倉に本格的に読ませるのは次話以降。
- 仮理由は脚本上のメモであり、真相説明ではないと本文の温度で分かるようにする。
