# 文体相談

## 005 最小置換

- 「美咲の制服ではなく、薄いニットに黒いパンツ。」
  - 「美咲の衣装ではなく、薄いニットに黒いパンツ。」
- 「画面の中では、美咲が白いものを胸に抱いたまま止まっている。」
  - 「画面の中では、美咲が白い封筒を胸に抱いたまま止まっている。」
- 冒頭の「美咲の手元に、白いものが残っている」も「白い封筒」でよい。
- 後半の「まだ渡されていないもの」は余韻として残す。

## 008 時系列

冒頭の黒い上着の段落後に、別日の駅前ロケでエキストラだったことを置く。

```text
先日の現場も、指定は黒系の上着だった。駅前で改札を抜ける通行人のひとり。名前のある役ではなく、雨避けの屋根の下を何度か歩いただけだった。魔王という言葉は、その日はまだどこにもなかった。
```

## 008 電話まわり

高瀬の台詞で、山野の画面映えを業務連絡として淡々と伝える。本人の内心で「画面映えする」とは言わせない。

採用候補:

```text
『先日の駅前ロケ素材を確認したところ、山野さんが画面の端に入った時、そこだけ視線が止まるという話が監督と演出側から出ています。
```
