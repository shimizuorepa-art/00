# 第4話/第5話 カオス差し込み案

## 目的

第4話は、悠人に会えた安堵より先に「馬車で来た」の一言で駅前の日常が破綻し始める形へ寄せる。第5話は、破綻が撮影現場の段取りへ反転したあとも、原作者のチャチャ、制作都合、出演者の戸惑いが同時に見える形へ厚くする。

校了品側と制作ラウンド側の完成稿は一致しているため、親エージェントが反映する場合は以下の差し込みを両方へ同内容で適用するのがよい。

## 第4話 004_馬車できた

### 1. 「駅前が折れた」を具体化する

現行の「安心するより先に、駅前が折れた。」は抽象度が高く、何が起きたのか少し掴みにくい。ここは、読者の目に見える動きへ置き換える。

置換案:

```md
美咲が「会えた」と思うより早く、バス停の列が一台ぶんだけ裂けた。
```

または、もう少し映像寄りにするなら:

```md
美咲が「会えた」と思うより早く、ロータリーの人の流れが、一台ぶんだけ馬車の形にへこんだ。
```

おすすめは前者。すぐ後ろの「傘の列だった」につながりやすい。

### 2. バス/配車/馬車の逃げ道を悠人が潰す

現行の聞き間違いはすでに良いので、二度目の「馬車で来た」の前後で、バスと配車に逃げようとする美咲の認識をもう一段だけ強くする。

差し込み位置:

`「いや、配車だと思ったんだけど」` の直後。

差し込み案:

```md
配車。

それならまだ分かる。アプリで呼んだ車。タクシーの言い間違い。雨の音で「バス」が濁っただけ。美咲は一瞬で、現実に戻るための言い訳を三つ並べた。

悠人は、その三つを全部知らない顔で踏んだ。
```

その後に `「馬車で来た」` を置くと、会話のオチで現実が閉じる。

### 3. 群衆にファンタジーを混ぜる

第4話で馬車だけに焦点が行くと、異世界転生が「一台の異物」に収まりやすい。ユーザー要望どおり、中世ファンタジーが駅前の群衆へ雑にねじ込まれていく感じを足す。

差し込み位置:

`最初に見えたのは、馬車ではなかった。傘の列だった。` の直後。

差し込み案:

```md
その傘の列に、雨合羽ではない外套が混じっていた。

鉄の肩当てをつけた男が、ビニール傘の下でスマートフォンを見ている。耳の長い女が、濡れた髪を指で払って、何事もない顔で信号を待っている。犬のように長い鼻面の人影が、コンビニの明かりを避けるように少し首を傾けた。

槍を持った人がいた。

弓を背負った人もいた。

どちらも雨に濡れているのに、誰も警察を呼ばない。誰も写真を撮るふりをしない。ただ少しだけ避ける。傘の列は、最初からそういう人たちも歩く場所だったみたいに、半歩ずつ形を変えていく。
```

この差し込みで、馬車が出る前に「駅前の交通」ではなく「駅前の人間のカテゴリ」が壊れる。

### 4. 馬の存在は見せるが、少しだけ不確かにする

第5話で「本物の馬は手続きが間に合わないので合成」という現場都合を出すなら、第4話では美咲視点で本物に見える程度に留めると面白い。読者は本物の馬を見た気になるが、第5話で「あれ合成予定なの？」と現場のズレが出る。

差し込み位置:

`ロータリーの端に、それはいた。` の後。

差し込み案:

```md
車体の前で、馬の首らしい黒い輪郭が一度だけ揺れた。鼻先から白い息が出たように見えた。雨の粒がそこだけ細かくはねて、美咲は本当に生きていると思ってしまった。
```

「らしい」「ように見えた」を入れることで、後の合成話と矛盾しにくい。

### 5. 封筒を触る理由をここで回収する

美咲が第1話から封筒を触り続けているなら、第4話の破綻に合わせて、手紙が悠人を呼んだのかもしれないという感触を入れると情緒が出る。

差し込み位置:

`雨粒が手首へ落ちる。鞄の口は閉じている。封筒は見えない。` の後。

差し込み案:

```md
あれは、最初に作ったやつだった。

何度も書き直して、いちばんきれいな言葉だけを残したはずなのに、捨てられなかった一通。消した言葉の筆圧が、紙の裏にまだ残っている。書かなかったことのほうが強く残って、美咲の指を内側から呼んでいる。

だから、触ってしまう。

悠人の声が手紙から来たのか、手紙が悠人をここまで呼んだのか、美咲にはもう分からなかった。
```

「最初に作ったやつだ」を地の文へ変換し、筆圧に呼ばれる感じを入れる案。

### 6. 魔王は名札で雑に確定させる

現行の「魔王みたいだ」は良いが、ユーザー要望はもっと乱暴でよい。現場が雑に作った小道具が、劇中では意味を持ってしまう感じにする。

置換対象:

`名札は雨粒で読めない。`

置換案:

```md
胸の名札には、白い養生テープに黒いマジックで「魔王」とだけ書いてあった。店長の名前ではなかった。
```

このあと現行の「魔王みたいだ」を少し調整して、

```md
魔王みたいだ、ではなかった。魔王と書いてある。
```

まで言い切ると、雑さと怖さが同時に出る。

## 第5話 005_カット

### 1. ファンタジー群衆をエキストラへ戻す

第4話で騎士や狼男を混ぜた場合、第5話冒頭でそれらが「役者」「衣装」「段取り」に戻ると反転が効く。

差し込み位置:

`透明傘の群れが、通行人からエキストラへ戻っていく。` の後。

差し込み案:

```md
肩当てをつけた男が、スタッフに渡されたビニール袋へ濡れた手甲を入れる。耳の長い女は、付け耳の根元を押さえながらメイク部を探していた。狼男のマスクを半分だけ外した人が、「息、白くなってます？」と誰かに聞く。

槍と弓は、カラーコーンの横にまとめて立てかけられていた。駅前の禁止事項の隣に、冒険者の荷物みたいなものが普通に置かれている。
```

ここで「劇中では本物っぽかったもの」が現場の備品へ戻る。

### 2. 魔王名札の雑さを現場側で回収する

差し込み位置:

`「魔王店長役の山野さん、そのまま待機で」` の後。

差し込み案:

```md
山野の胸では、「魔王」とだけ書かれた養生テープが雨で少しめくれていた。

「名札、これでいいんですか」

山野が小さく聞く。

「分かればいいです。魔王なんで」

誰かが答えた。雑な答えだったが、雑なぶんだけ原作に近い気もした。
```

「名札に魔王って書いとけ」の現場感をそのまま出せる。

### 3. 馬不在は手続きと安全で押し返す

現行の「手続きが間に合わないんで。あとで合成します」は良い。もう少し、雨のロータリー・一般交通・動物手配の無理さを入れると考証の匂いが出る。

置換案:

```md
「ところで、本物のウマさんは今日いないんですか？」

鳴海が、馬車の前方を探すように言った。

高瀬は一度だけ目を閉じた。

「この雨で、駅前ロータリーで、一般車もバスも動いてます。道路使用と動物手配と安全導線が、今日の今日で間に合うわけないです。あとで合成します」

誰のせいで、と誰かが言いかけた。
```

「誰かのせいで」は言い切らず、現場の空気で伝える。

### 4. 原作者のチャチャをもう一段悪くする

既存の雨粒が魚、ドラゴン、勇者設定のやり取りは良い。鳴海が楽しそうに現場を追い詰め、高瀬と真鍋が事務的に潰す形を補強する。

追加案:

```md
「ちなみに、このあと雨粒が魚になります」

鳴海が、天気の話みたいに言った。

高瀬は無線を持ったまま、空を見ない。

「合成です」

「ドラゴンが降りてきます」

「合成です」

「そこで悠人さんが、自分が勇者だった設定を開示して、魔王店長さんと一触即発に」

「そこも今日はやりません」

「え、魔王店長さんはいますよ」

鳴海はコンビニの入口を指した。山野は胸の「魔王」を押さえたまま、聞こえないふりをした。
```

この方が、原作者が映像化をからかっている感じが強く出る。

### 5. 出演者の戸惑いは「恋愛映画の顔」からずらす

佐伯と朝倉は、さっきまで恋愛映画の芝居をしていた人たちとして、次ページの特撮展開に困ると良い。

差し込み/置換案:

```md
佐伯蓮は次のページを見て、しばらく動かなかった。

「俺、恋愛映画って聞いてたんですけど」

「私もです」

朝倉が横から台本を覗く。濡れたページの端に、赤字で「魔法を使う時の手の動きはアドリブ」と書かれていた。

「アドリブで魔法？」

佐伯が言う。

「手って、こうですか」

朝倉が少しだけ指を開く。メイク部がタオルを持ったまま固まり、佐伯は真面目にそれを見てしまった。二人とも笑っていない。笑うには、台本が本気すぎた。
```

出演者がバカにせず本気で困る方が、あとで本気の芝居になる余地が残る。

## 優先反映順

1. 第4話の「駅前が折れた」置換。
2. 第4話のファンタジー群衆差し込み。
3. 第4話の封筒理由差し込み。
4. 第4話/第5話の魔王名札回収。
5. 第5話の馬不在=手続き/安全/合成の強化。
6. 第5話の原作者チャチャと出演者戸惑いの増量。

この順で入れると、カオスは露骨になるが、話の主軸は「悠人に会えたと思った瞬間、現実のほうが壊れる」から外れない。
